# Prompts

Reusable task prompts. Use them, don't freestyle.

## Use

1. open prompt
2. fill `{{vars}}`
3. follow steps
4. run [`validate-repo.md`](validate-repo.md)

## Index

| task | prompt |
|------|--------|
| add source | [`add-source.md`](add-source.md) |
| bump source | [`bump-source.md`](bump-source.md) |

## Shape

Each prompt should have:

- purpose
- inputs
- preconditions
- steps
- output
- post-checks
